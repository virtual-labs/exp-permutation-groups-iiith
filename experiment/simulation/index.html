Before you start:
  • 15-puzzle is a type of Permutation puzzle that arranges the first sixteen natural numbers in a four by four grid with each number representing their numbered tile and sixteen being empty space.
  • Clicking on generate will create a random solvable 15 puzzle arrangement and its current permutation as displayed in the simulation.
  • To solve the 15-puzzle, you move tiles adajacent to the empty space, into the empty space by clicking on them. Each move represents a two cycle of the permutation which is multiplied to the current permutation
  • The goal of the puzzle is arrange the numbered tiles in increasing order. So the numbers increase left to right and then following the same oreder in the next row.
  • This should result in the current permutation finally being the identity permutation
  • If at any point you get stuck, the hint button will provide you with the next step to take in the algorithm to solve the puzzle.
